# simple-crud-api
How to run project
Clone repo locally : git clone https://github.com/Dewlar/simple-crud-api.git

- Go to folder with project: `cd simple-crud-api`

- Switch to develop branch: `git checkout develop`

- Change file `.env.example` to `.env`

- Intsall all dependencies `npm install`

Scripts
- To run in development mode: `npm run start:dev`

- To run in production mode: `npm run start:prod`

- To run load balancer: `npm run start:multi`

- To run tests: `npm run test`
